API name API Description Activate
Postcode Lookup Returns the county, place name and co-ordinates associated with the Country code and Postcode provided.

A Regex string is used to clean input data, removing spaces and unwanted characters. Postcode variants such as "SW1A0AA", "SW1A 0AA" and "SW!"£1A 0$£!AA" are all successful.

For ease of use the Longitude and Latitude generated from this search will automatically populate the weather and timezone search fields below so that the user does not have to enter it manually.

All APIs on this page produce a short "searching..." message while attempting to contact their respective server.

If any search criteria is incorrect an appropriate error message is displayed on the page and logged to console.




Find Nearby Weather Returns the temperature, clouds, weather, humidity and wind speed based on the Longitude and Latitude provided.


Timezone Returns the timezone, sunrise, sunset and current time for a region based on the longitude and latitude provided.
